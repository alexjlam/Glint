How to deploy an image onto a cloud:

1. upload image to Glint
Glint Images page under 'Add new image'
a) enter name for image
b1) upload image file
or b2) enter downloadable image address
c) [Upload Image]

2a. upload Openstack site
Openstack site -> Access & Security -> API Access tab
DOWNLOAD Openstack RC File
Glint Sites page under 'Add new Openstack site'
a) enter name and your password for the site
b) upload the Openstack RC File that was downloaded
c) [Upload Sites]

or 2b1. upload EC2 Site
Security Credentials in IAM Management Console on Amazon Web Services
GET Account Number, Access and Secret Keys
DOWNLOAD certificate and private key files
Glint Sites page under 'Enter EC2 credentials'
a) enter account number, access key, and secret key
b) upload certificate and private key that was downloaded
c) [Save EC2 Credentials]

2b2. bundle image
* in order to deploy onto EC2 site you must bundle your image first
* you can only bundle images that are from image files, not image addresses
Glint Images under 'Bundle images'
a) select your image you want to [Bundle]

3. deploy image to site
a) under the image you want to deploy select to site
* green sites are sites that the image is on/will be on after saving
* red sites are sites that the image in off/will be off after saving
b) once done making changes [Save]
